# Discord Ranked Bot

Discord Bot für Team-Management mit Google Sheets Integration.

## Features
- Teams aus Google Sheets in Discord posten
- Automatische Member-Verschiebung in Voice Channels
- Confirm-Button für Team-Bestätigung
- Webhook-Integration mit Google Sheets

## Setup

1. Clone das Repo
2. `npm install`
3. `.env` Datei erstellen (siehe `.env.example`)
4. `npm run build`
5. `npm start`

## Deployment auf Railway

1. GitHub Repo mit Railway verbinden
2. Umgebungsvariablen in Railway eintragen
3. Deploy!
